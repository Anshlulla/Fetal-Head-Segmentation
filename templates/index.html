How does it work?

  • Dataset Collection

    Fetal head ultrasound images are collected, including datasets for PSFHS (Persistent Second Fetal Head Station) and fetal brain abnormalities.

  • Preprocessing

    Images are resized, normalized, and enhanced using techniques like CLAHE, wavelet denoising, non-local means, bilateral filtering, and entropy filtering.

  • Segmentation

    Segmentation is performed using architectures like SegNet, UNet and AttentionUNet integrating spatial and frequency domain features.

  • Classification

    Segmented images are classified into normal or abnormal categories using models like ResNet with SE and CBAM attention mechanisms.

  • Explainability

    GRAD-CAM is used to visualize attention maps for model interpretability, highlighting critical regions in the ultrasound.

  • Fine Tuning

    Models are fine-tuned through domain adaptation, freezing/unfreezing layers, recalibration, and progression-based tuning using head-pelvis distance.

  • Final Output

    The system predicts delivery type based on head position (floating, engaged, deep engagement) and rotation (OA, OP, OT) for PSFHS intrapartum assessment.

Frequently Asked Questions

We use fetal head ultrasound datasets, PSFHS (Persistent Second Fetal Head Station), and fetal brain abnormality datasets to train and validate segmentation and classification models.

Preprocessing involves resizing, normalization, CLAHE enhancement, wavelet denoising, non-local means filtering, bilateral filtering, and entropy-based filtering to improve image clarity and segmentation performance.

We use SegNet, UNet and Attention UNet for segmentation. For classification, we implement ResNet-50 with SE/CBAM attention modules.

The system predicts whether the development is normal or abnormal. For PSFHS cases, it determines head position (e.g., Floating, Engaged) and head rotation (OA, OP, OT), supporting intrapartum delivery type assessment.

We use Grad-CAM visualizations to interpret the decision-making regions of the classification model, helping clinicians trust and verify predictions from the deep learning system.
